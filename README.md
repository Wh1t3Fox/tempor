# tempor

### Usage
```
➜ make
Option			Description
build                   Build Docker Containers
build-nc                Build Docker Containers w/out Cache
clean                   Cleanup VPS and Files
run                     Run Terraform
show                    Show VPS Options
test                    Live Testing with Terraform
destroy                 Destroy VPS'
```

### Setup

```
➜ cat api_tokens.mk
do_token=
#li_token=
#vultr_token=
```
\# to comment out API token
